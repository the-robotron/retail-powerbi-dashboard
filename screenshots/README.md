# Dashboard Screenshots

This folder contains screenshots of the **Retail Sales Power BI Dashboard**.
Add your screenshots here after building the dashboard in Power BI Desktop.

---

## How to Add Screenshots

1. Build the dashboard following instructions in `pbix/retail_dashboard_info.md`
2. In Power BI Desktop, use **File → Export → Export to PDF** or take screenshots of each page
3. Save images in this folder with descriptive names
4. Update the table below with your screenshot filenames

---

| Page | Filename | Description |

---

## Recommended Screenshot Dimensions

- **Resolution:** 1920 x 1080 (Full HD)
- **Format:** PNG (preferred) or JPEG
- **File size:** Keep under 2MB per image for fast GitHub loading

## Tips for a Great Portfolio Dashboard

- Use consistent colors across all pages (stick to 2-3 primary colors)
- Add a company logo placeholder in the top-left corner
- Include a last-refreshed date/time indicator
- Add tooltips to all visuals for better interactivity
- Use bookmarks to create navigation buttons between pages
- Test on both desktop and mobile layout views
